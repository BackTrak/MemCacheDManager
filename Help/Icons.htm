Icons and Definitions

The following table is a list of Icons and their definitions.

The item is operating normally.
A server element is defined but doesn't have any instances assigned to it yet. Right click on the server element and add an instance to it to install MemCacheD on the server.
The server is running a version of MemCacheD that is not the same as the one MemCacheD Manager is configured to use. Right click on the server to update the server to the latest version of MemCacheD.
The MemCacheD service is running, but is not communicating with MemCacheD Manager. This may indicate a miss configured network, or blocked MemCacheD port.
The MemCacheD service is stopped on the server.
There was an error trying to get the MemCacheD service status from the server. This error can be caused by an authorization error, WMI provider error, or network connectivity.
