Upgrading to Your Version of MemCacheD

MemCacheD manager comes bundled with the current release version of MemCacheD. Over time newer versions of MemCacheD will be released, or you may find it necessary to downgrade to a previous version. MemCacheD Manager makes it easy to do so.

 

 

Using the options tab, you can change the MemCacheD binary that MemCacheD Manager deploys to remote servers from the Options tab. Simply click the browse button to the right of the MemCacheD Binary text box to browse to the binary you would like to deploy. The version of the binary appears in the MemCacheD Version text box.

 

 

After you change the version, the Status and Configure tabs will indicate which MemCacheD services require an update to make them match your version.

 

 

Simply right click on each server and select Redeploy / Update MemCacheD and your specified version will be copied to the remote machine, and the MemCacheD service will be restarted.
