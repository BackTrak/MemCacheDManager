Creating a New MemCacheD Instance

Before you can create a MemCacheD instance on a server, you should first define the server in the MemCacheD manager. Please see this section if you do not already have a server defined.

MemCacheD manager can control many servers, with each server running one or more instances (a configuration of MemCacheD running as a Windows service).


You can add instances of MemCacheD to your server by right-clicking on the server's name and then selecting "Add New Instance".

Specify the Instance Name, and configure any of the MemCacheD parameters to tune MemCacheD to your environment, and then click Apply. MemCacheD will then be copied to the server if it's already present, and then installed as a Windows service.

MemCacheD is now running on your server.
