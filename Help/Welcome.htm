Introduction

This utility is designed to reduce your administration tasks when deploying and maintaining a running MemCacheD environment. It has the following features:
  • Able to remotely create, edit and control MemCacheD as a windows service.
  • Use Windows account credentials for impersonation so that you can deploy to multiple Windows domains from a single administration console.
  • Uses simple files so that configuration can be shared between administrators.
  • Give a simple status view of all the MemCacheD services in your enterprise and some basic health metrics.
  • Enables simple parameter setting and viewing of MemCacheD settings in your enterprise.
  • Can discover manually installed MemCacheD services on a given server.
  • Can generate .Net configuration sections for .Net MemCacheD APIs for easy integration.
  • Allows you to detect when a server is not running your version of MemCacheD.
  • Allows you to perform upgrades of MemCacheD to a particular version remotely.
  • Use multiple configuration files for your various environments. 

 

The utility has some caveats that you should be aware of:

  • Does not work with non-Windows MemCacheD instances.
  • Requires WMI to be installed and running on all machines that you would like to administer.
  • Requires administrative privileges on any machine that you would like to administer. (Impersonation is supported).
